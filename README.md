A fully decentralized social platform with post, comments, likes, retweets and all interactions being on chain, building a way for communities to organize themselves in a fully decentralized way.
We allow (NFT) communities to build decision layers through built-in governance, so that they can organize and govern their business with their community in a collaborative way, including microbounties for activations.

Use case:
- have conference starting
- create proposal for NFT community to launch activation there
- discuss with community
- discuss pricing
- apply to get treasury funding for that
- if confirmed, start activation
- create microbounty for companies to supply with food or for hackathon (also usuable by DeSci communities to launch peer-review articles as microbounties)
- show microbounty on hup job board
